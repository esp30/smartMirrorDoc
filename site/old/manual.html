If you work at or for the canteen...

You can interact with our system using the canteen control interface.


1. First, you'll need to create an account using with our app

2. Add your recipes

 Recipes have 1 or more ingredients that can be chosen from a predefined list of foods.

3. Create the menu of the day

Canteen workers can create and edit the menu of the day by adding or removing previously created recipes.
